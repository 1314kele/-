本文档详细说明校园二手交易平台的测试方案，包括性能测试、功能测试、单元测试和自动化测试的完整流程。


## 📊 性能测试

### 测试内容

1. **数据库性能测试**
   - 数据库连接时间
   - 查询性能（100个商品）
   - 写入性能（10个商品）

2. **API接口性能测试**
   - 用户登录时间
   - 首页加载时间
   - 商品列表页时间
   - 搜索功能时间

3. **并发性能测试**
   - 10个线程，每个线程20个请求
   - 平均响应时间
   - 最大/最小响应时间
   - 每秒请求数

4. **内存使用测试**
   - 测试前后内存使用量
   - 内存增加量

### 运行性能测试

```bash
# 单独运行性能测试
python performance_test.py
```

### 性能基准指标

| 测试项目 | 优秀指标 | 良好指标 | 需要优化 |
|---------|---------|---------|---------|
| 数据库连接时间 | < 0.05s | < 0.1s | > 0.2s |
| 查询100个商品 | < 0.2s | < 0.5s | > 1s |
| 用户登录时间 | < 0.5s | < 1s | > 2s |
| 首页加载时间 | < 1s | < 2s | > 3s |
| 平均响应时间 | < 0.5s | < 1s | > 2s |
| 每秒请求数 | > 50 | > 20 | < 10 |

## ⚙️ 测试配置

### 配置文件

`test_config.py` 包含完整的测试配置：

```python
TEST_CONFIG = {
    'TEST_DATA': {
        'NUM_USERS': 100,        # 测试用户数量
        'NUM_PRODUCTS': 200,      # 测试商品数量
        'NUM_CATEGORIES': 10,     # 测试分类数量
    },
    'PERFORMANCE_TEST': {
        'CONCURRENT_THREADS': 10, # 并发线程数
        'REQUESTS_PER_THREAD': 20,# 每个线程请求数
    }
}
```

### 自定义配置

可以修改 `test_config.py` 文件来自定义测试参数：

1. 调整测试数据规模
2. 修改性能测试参数
3. 配置测试报告格式
4. 设置邮件通知

## 🔍 测试用例设计

### 测试优先级

| 优先级 | 测试模块 | 测试重点 |
|-------|---------|---------|
| 关键 | 用户认证、交易流程、支付安全 | 核心业务流程 |
| 高 | 商品管理、搜索功能、消息系统 | 主要功能模块 |
| 中 | 用户行为分析、推荐系统、安全日志 | 辅助功能模块 |
| 低 | 界面样式、性能优化、辅助功能 | 用户体验优化 |

### 测试数据策略

1. **边界值测试**
   - 价格边界（0元、极高价格）
   - 数量边界（空列表、大量数据）
   - 字符边界（空字符串、超长字符串）

2. **异常情况测试**
   - 无效输入
   - 权限不足
   - 网络异常
   - 数据库异常

3. **并发测试**
   - 多用户同时操作
   - 资源竞争
   - 死锁检测

## 🛠️ 故障排除

### 常见问题

1. **数据库连接失败**
   ```bash
   # 检查数据库文件
   ls -la db.sqlite3
   
   # 重新创建数据库
   python manage.py migrate
   ```

2. **依赖包缺失**
   ```bash
   # 安装缺失包
   pip install psutil requests
   ```

3. **测试超时**
   - 检查网络连接
   - 增加测试超时时间
   - 优化测试数据规模

4. **内存不足**
   - 减少测试数据量
   - 关闭其他应用程序
   - 增加系统内存

### 调试技巧

1. **查看详细日志**
   ```bash
   python automated_test_runner.py 2>&1 | tee test.log
   ```

2. **单独运行失败测试**
   ```bash
   python -m pytest path/to/failed_test.py -v
   ```

3. **性能分析**
   ```bash
   python -m cProfile performance_test.py
   ```
