# 校园二手交易平台数据生成脚本使用说明

## 脚本功能

这个脚本会为您的校园二手交易平台生成真实的测试数据，包括：

- **用户数据**：10个测试用户账户
- **分类数据**：5个商品分类（电子设备、书籍资料、生活用品、运动器材、学习用品）
- **商品数据**：50个真实商品，每个分类10个商品
- **交易数据**：15条交易记录
- **评价数据**：20条用户评价
- **收藏数据**：30条收藏记录
- **消息数据**：25条用户消息
- **用户行为**：100条用户行为记录
- **知识库**：8条知识库条目
- **用户偏好**：用户对分类的偏好数据
- **商品图片**：自动下载真实的商品图片

## 使用步骤

### 1. 确保环境准备就绪

确保您已经：
- ✅ 安装了Python 3.7+
- ✅ 创建了Django虚拟环境并激活
- ✅ 安装了必要的依赖包

### 2. 安装必要的Python包

在项目根目录下运行：

```bash
pip install requests pillow
```

### 3. 创建media目录

确保项目中有media目录用于存储图片：

```bash
cd demo
mkdir -p media/products
```

### 4. 运行数据生成脚本

在项目根目录下运行：

```bash
cd demo
python generate_realistic_data.py
```

## 常见问题

### Q1: 脚本运行失败，提示找不到Django设置？

**A**: 确保在项目根目录下运行脚本，并且Django环境变量设置正确。

### Q2: 图片下载失败？

**A**: 检查网络连接，或者注释掉图片下载部分，使用占位符图片。

### Q3: 数据库连接错误？

**A**: 确保已经运行了数据库迁移：
```bash
python manage.py migrate
```

### Q4: 如何重新生成数据？

**A**: 可以先清空数据库，然后重新运行脚本：
```bash
python manage.py flush
python generate_realistic_data.py
```

### Q5: 如何修改数据数量？

**A**: 编辑脚本中的以下参数：
- `USERS`：修改用户数量
- `CATEGORIES`：修改分类和商品数量
- 函数调用中的数字：修改交易、评价、消息等数量
